# Markdown文件基本语法

## 如何写标题

```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```



---



## 粗体与斜体

```markdown
**粗体**
*斜体*
***粗体与斜体***
```



---



## 删除线

```markdown

```
